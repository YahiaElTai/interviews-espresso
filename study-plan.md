# Study Plan

## Courses & Books

Study each course/book just before working on its related topic file(s).

1. **Hussein Nasser — Fundamentals of Database Engineering** (Udemy, 26h)
   → before `02-data/01-databases`, `02-data/02-postgresql`

2. **NeetCode YouTube + LeetCode** (ongoing)
   → before `05-foundations/01-data-structures-and-algorithms` — start here, continue daily alongside everything else

3. **Hussein Nasser — Fundamentals of Network Engineering** (Udemy, 15h)
   → before `05-foundations/02-networking-and-protocols`

4. **Hussein Nasser — Fundamentals of Operating Systems** (Udemy, 21h)
   → before `05-foundations/03-operating-systems-and-linux`, `05-foundations/04-concurrency-and-parallelism`

5. **Alex Xu — System Design Interview Vol 1** (book, fast read)
   → before `07-system-design/` files

6. **Designing Data-Intensive Applications — Martin Kleppmann** (book, slower/deeper)
   → before `07-system-design/` files, `06-architecture/04-event-driven-architecture`, `06-architecture/07-scaling-and-reliability`

7. **The Software Engineer's Guidebook** (book)
   → before `10-behavioral/01-leadership-and-behavioral`
